# 📖 วิธีใช้งาน

ทำตามขั้นตอนนี้เพื่อสร้างลิสต์เพลงโปรด 3 อันดับของคุณ

## ขั้นตอนหลัก

1. เข้าเว็บ [mytop3tracks.online](https://www.mytop3tracks.online/)
2. กดปุ่ม **\+** ในช่อง 01, 02 หรือ 03
3. วางลิงก์เพลงจาก YouTube หรือ YouTube Music
4. กด **Done** เพื่อบันทึกเพลงในช่องนั้น
5. ใส่เพลงให้ครบ 3 ช่อง
6. กด **Generate Top 3 Tracks** เพื่อสร้างผลลัพธ์
